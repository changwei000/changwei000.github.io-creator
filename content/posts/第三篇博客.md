# 步骤如下所示：
1. 官网下载并安装hugo，利用```hugo version```命令检查你只否安装成功；
2. 创建目录，执行 ```hugo new site quickstart ```( 其中，quickstart改成自己的仓库名【小写的用户名.github.io-creator】)；
3. 添加主题,利用命令```cd quickstart``` 进入刚刚创建的目录, 接下来只需要机械地直接执行这三句命令：
   
    ```git init ```

    ```git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke```

    ```echo 'theme = "ananke"' >> config.toml```
4. 添加新的博客,执行
```hugo new posts/my-first-post.md```（其中，my-first-post改成自己的博客主题）
快去自己的vscode编辑博客内容吧（注：想要发布就把draft改成false状态）；
5. 运行命令```
hugo server -D```，点击所给链接 https://localhost.1313 就可以本地预览自己的博客界面喽；
6. 更改主题，在vscode里面用ctrl+p打开搜索
config.toml，将里面的title改成自己的主题即可，语言也可以改成“zh-Hans”；
7. 将public单独拎出来，运行命令```hugo cd public```创建public目录；在第一步骤创建的目录中，创建.gitignore文件并在里面编辑/public/内容,(旨在让当前这个目录忽略public)；在public目录下,执行 ```git init``` 来创建public的.git文件，再将通过```git add .```和```gc```提交public；
8. 去你的github新建一个仓库，仓库名必须是【用户名.github.io】，再分别复制两行代码来运行，刷新github；
9. 打开settings找到GitHub Pages点击它所给的地址即可看到你的博客啦，当然到此你的博客就可以在你的github上看了！！！
