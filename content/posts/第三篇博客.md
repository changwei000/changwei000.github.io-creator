# **步骤**

## $\color{#FF00FF}{1.}$$\color{#FF0000}{官网}$$\color{#FF7D00}{下载}$$\color{#FF0000}{并}$$\color{#00FF00}{安装}$$\color{#0000FF}{hugo}$

   
   ```go
   运行hugo version命令
   ```
   [注] 检查你只否安装成功；
## $\color{#FF00FF}{2.}$$\color{#FF0000}{创建}$$\color{#FF7D00}{目录}$
   ```go
   运行hugo new site quickstart命令
   ```
   [注] 其中，quickstart改成自己的仓库名——
   小写的用户名.github.io-creator
## $\color{#FF00FF}{3.}$$\color{#FF0000}{添加}$$\color{#00FF00}{主题}$
 
* ```go
   运行cd quickstart命令
   ```
   [注] 进入上一步创建的目录
     
* 接下来只需要机械地直接执行这三句命令：
    ```go
    git init

    git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

    echo 'theme = "ananke"' >> config.toml

## $\color{#FF00FF}{4.}$$\color{#FF0000}{添加}$$\color{#FF7D00}{新的}$$\color{#FF0000}{博客}$
   
   ```
   运行hugo new posts/my-first-post.md命令
   ```    
   [注] 其中，my-first-post改成自己的博客主题
## **$\color{#FF0000}{5.}$$\color{#FF7D00}{本地}$$\color{#FF00FF}{预览}$**
 
```
运行hugo server -D命令
```
[注] 点击所给链接 https://localhost.1313 就可以本地预览自己的博客界面喽

## $\color{#FF0000}{快去}$$\color{#FF7D00}{编辑}$$\color{#FF0000}{你的}$$\color{#00FF00}{第一篇博客}$$\color{#0000FF}{吧}$$\color{#FF00FF}{!!!}$ 
  
[注] 发布时，记得将$\color{#FF0000}{draft}$改成$\color{#FF0000}{false}$状态
